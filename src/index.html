Welcome to my product management system CRUD application. Please find below the relevant APIs to test the application:-

  1. Create:
    • Request Method: POST
    • Request API: https://product-mgt-sys.onrender.com/api/product/create
    • Request Body: {"productName": "apple"}
  2. Retrieve Single Product:
    • Request Method: GET
    • Request API: https://product-mgt-sys.onrender.com/api/product/retrieve/:productId
  3. Retrieve All Products:
    • Request Method: GET
    • Request API: https://product-mgt-sys.onrender.com/api/product/retrieve
  4. Update:
    • Request Method: PUT
    • Request API: https://product-mgt-sys.onrender.com/api/product/update/:productId
    • Request Body: {"productName": "apple cream"}
  5. Delete:
    • Request Method: DELETE
    • Request API: https://product-mgt-sys.onrender.com/api/product/delete/:productId
